# API Documentation

See OpenAPI docs at `/docs` when running the FastAPI server.

## Main Endpoints
- `/api/v1/auth/login` (POST)
- `/api/v1/auth/register` (POST)
- `/api/v1/users/me` (GET)
- `/api/v1/profiling/label` (GET)
- `/api/v1/schedule/generate` (POST)
- `/api/v1/progress/log` (POST)
- `/api/v1/progress/predict` (GET)
- `/api/v1/stress/log` (POST)
- `/api/v1/motivation/log` (POST)
- `/api/v1/motivation/classify` (GET)
- `/api/v1/community/peers` (GET)
- `/api/v1/group/match` (POST)
