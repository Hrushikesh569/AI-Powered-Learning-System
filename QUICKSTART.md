# Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Navigate to Project
```bash
cd d:\Summer-Intership\learning-scheduler
```

### Step 2: Install Dependencies
```bash
npm install
```

This will install:
- React 18
- React Router
- Tailwind CSS
- Recharts
- Framer Motion
- Lucide React
- Vite

### Step 3: Start Development Server
```bash
npm run dev
```

Then open: **http://localhost:5173**

---

## 📱 Test the Application

### Navigation Flow
1. Start at **Landing Page** (`/`)
2. Click "Get Started" → **Register** (`/register`)
3. Fill form → **Profiling** (`/profiling`)
4. Complete questionnaire → **Dashboard** (`/dashboard`)
5. Explore:
   - **Analytics** (`/analytics`) - View charts
   - **Community** (`/community`) - Browse groups
   - **Profile** (`/profile`) - Edit settings
